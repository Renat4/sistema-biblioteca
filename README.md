# Sistema de Biblioteca

Este projeto foi desenvolvido para armazenar e buscar livros utilizando Python.

## Estrutura de Dados

A estrutura de dados utilizada foi o dicionário (dict)

A escolha foi feita porque o dicionário permite armazenar os livros utilizando um ID único como chave.

Isso torna a busca muito eficiente, pois é possível acessar diretamente o livro pelo ID, sem precisar percorrer todos os elementos.

A complexidade da busca em um dicionário é O(1), ou seja, muito rápida, mesmo com muitos dados.

## Funcionalidades

- Adicionar livro
- Buscar livro por ID
